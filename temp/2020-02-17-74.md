# SPA (Single Page Application)

SPA에서는 하나의 웹 페이지가 하나의 어플리케이션으로 동작하는 것이다.  
따라서 웹 페이지를 띄우는 것은 어플리케이션을 Boot 하는 것이고,  
웹 페이지를 닫는 것은 Shutdown 하는 것,  
그리고 페이지를 새로고침 하는 것은 Reboot 하는 것이라고 생각하면 된다.
